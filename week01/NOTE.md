# 学习笔记

本周学习了学习方法和如何构建知识体系。


有几个方面的收获：
1. 了解如何用整理法和追溯法去构建自己已有的知识体系和想要去扩展的知识，将精力更优化地分配。
2. 自己接触前端并不深，也没有很好的工作环境去深入了解前端技术，以前总是对此非常忧虑，不知道自己能对前端掌握的什么程度。
现在有了整理法，那么就能快速了解一项技术的大体全貌，判断是否符合自己的需求，比如Electron。然后用追溯法去深入了解其中关键部分的原理，达到举一反三的目的。
